## Odbg shortcut
F2 : Break Point
F7 : Step Into (상세하게 분석 : 반복, 함수분석시 사용)
F8 : Step Over
F9 : Run
ctrl + f2 : Restart

## tip
c로 assembly로 변환한 code

### main 함수 찾기
1. GetCommandLine찾기
2. 인자가 2개에서 최대 4개까지 가지고 있는 함수 찾기 = main함수
3. 함수 클릭 -> Enter 
4. F2(Break point) 
5. F9눌러서 main까지 접근 후에 분석시작
